# Pomodoro Timer con Vite

Proyecto inicial de Vite con JavaScript para una aplicación Pomodoro.

## Estructura del Proyecto

```
pomodoro/
├── src/
│   ├── main.js      # Punto de entrada de la aplicación
│   ├── app.js       # Lógica principal de la app
│   └── styles.css   # Estilos
├── index.html       # HTML principal
├── vite.config.js   # Configuración de Vite
└── package.json     # Dependencias del proyecto
```

## Instalación

```bash
npm install
```

## Desarrollo

Inicia el servidor de desarrollo:

```bash
npm run dev
```

El servidor abrirá automáticamente en `http://localhost:5173`

## Build para Producción

Genera la versión optimizada:

```bash
npm run build
```

Los archivos compilados estarán en la carpeta `dist/`

## Preview de Producción

```bash
npm run preview
```

## Características

- ⚡ Configuración rápida de Vite
- 📦 ES modules nativos
- 🎨 CSS moderno incluido
- 🔄 Hot Module Replacement (HMR)
